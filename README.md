# php-bootstrap

This is a repository intended to serve as a starting point for bootstrap a project in PHP.
The idea is that you don't have to worry about the boilerplate.

- Best practices applied:
  - README.md
  - LICENSE
  - composer.json
  - composer.lock
  - phpunit.xml
  - .gitignore
